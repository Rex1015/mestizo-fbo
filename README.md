## 🚀 Setup

### 1. Add Your Logo
- Place your PNG logo in `assets/logo.png`
- Recommended size: 200px wide × 60px tall
- Transparent background preferred

### 2. Add Hero Video
- Place aviation video in `assets/hero-bg.mp4`
- Free sources:
  - https://www.pexels.com/search/videos/airplane/
  - https://coverr.co
- Recommended: MP4, under 15MB, 1920×1080

### 3. Set Up Email (Inquiry Form)
- Open `inquiry.html`
- Find this line:
  action="https://formsubmit.co/fbo@mestizodesangley.com"
- Replace with your actual email
- FormSubmit is FREE — no backend needed

### 4. Update Contact Info
- Search for "+63 917 123 4567" in all files
- Replace with your real phone number
- Search for "fbo@mestizodesangley.com"
- Replace with your real email

---

## 🌐 Deploy to GitHub Pages

1. Push all files to GitHub repo
2. Go to Settings → Pages
3. Source: Deploy from branch → main → / (root)
4. Your site: https://yourusername.github.io/mestizo-fbo/

## 🌐 Deploy to Hostinger

1. Log in to Hostinger hPanel
2. Go to File Manager → public_html
3. Upload ALL files (keep folder structure)
4. Visit your domain — done!
