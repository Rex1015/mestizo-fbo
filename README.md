# 🌿 Mestizo de Sangley FBO

Premier Fixed Base Operator at Sangley Point, Philippines.

## 🚀 Quick Deploy

1. **GitHub Pages**: Enable in Settings > Pages
2. **Hostinger**: Upload all files to public_html
3. **Video**: Add aviation video to `assets/bg-video.mp4`

- ✅ Green/White minimalist theme
- ✅ Video background hero
- ✅ Smooth animations & effects
- ✅ Dropdown services

- ✅ Fully responsive
- ✅ No external dependencies
